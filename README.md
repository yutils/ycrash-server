# ycrash-server，安卓崩溃异常捕获类Server端

用于接收ycrash安卓端的崩溃异常信息。


maven工程，数据库采用mysql，不包含HTML前端页面。



安卓端异常崩溃捕获看过来：[点击前往](https://github.com/yutils/ycrash)

配套的WEB前端页面请看过来：[点击前往](https://github.com/yutils/ycrash-web)
